Leetcode 516 Longest Palindromic Subsequence

题目描述

给定字符串s, 寻找该字符串中最长回文的子序列。假设最长长度为1000.

例: s = “bbbab” res = 4

解题思路

DP

$
首先介绍一种Time: O(n^2), Space: O(n^2) \
dp[i][j] 表示 s[i..j]中最长回文子串长度。 \
dp[i][j] = \begin{cases}
1 \qquad & i = j \
dp[i+1][j-1]+2 \qquad & s[i]==s[j]\
max(dp[i+1][j], dp[i][j-1]) \qquad & s[i]!=s[j]\
\end{cases} \
$
至此整个算法核心部分已经解决。
