Leetcode 121. Best Time to Buy and Sell Stock

题目描述

给定一个数组，它的第 i 个元素是一支给定股票第 i 天的价格。

如果你最多只允许完成一笔交易（即买入和卖出一支股票），设计一个算法来计算你所能获取的最大利润。

注意你不能在买入股票前卖出股票。

[7,1,5,3,6,4]
5

解题思路

由于只能买卖一次，因此每次都减去至今为止最小的买入价格就行。
[7, 1, 5, 3, 6, 4] - [7, 1, 1, 1, 1, 1]
至今最小可以使用一个变量 curMin 进行记录。
