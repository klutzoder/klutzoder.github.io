GCC

交叉编译命名规范：[arch]-[os]-[(gnu)eabi]
arch: 表示体系结构, ARM、MIPS
os: 表示目标操作系统
eabi: 嵌入式应用二进制接口

  1. arm-linux-gnueabi: 用于 ARM32 的 Liunx 系统编译。
  2. aarch-linux-gnueabi: 用于 ARM64 的 Linux 系统编译。
